# Mihael Pleško - Portfolio

A professional developer portfolio built for Mihael Pleško, a backend engineer specializing in distributed systems.

## Features
- **Vue 3 + Vite**: Lightning fast development experience.
- **Tailwind v4**: Next-generation utility-first CSS framework.
- **Custom Modern Cursor**: Inverts colors on hover.
- **Parallax 3D Scrolling**: Depth-based scrolling mechanics to add sophisticated feel.
- **Dichotomous Content**: Designed for both skimming recruiters ("At a Glance") and detailed readers ("Deep Dive").

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Design Iteration
Inspiration comes from Vercel's stark, minimalist aesthetics and modern high-end developer portfolios featuring smooth interaction mechanics and deep typography.
